Senior Frontend Engineer

GitLab · San Francisco, USA

🌐 Remote Full-Time 🕐 Any Timezone

About the Role

We're looking for a Senior Frontend Engineer to join our growing engineering team. You'll be responsible for building and maintaining user-facing features for GitLab's web platform, working closely with designers, product managers, and backend engineers. This is a remote-first role open to candidates worldwide.

Key Responsibilities

  • Develop and maintain high-quality, performant frontend features using Vue.js and modern JavaScript
  • Collaborate with UX designers to implement pixel-perfect, accessibility-compliant interfaces
  • Write comprehensive unit and integration tests to ensure code reliability
  • Participate in code reviews and contribute to engineering best practices
  • Optimize application performance including load times, rendering, and bundle size
  • Mentor junior engineers and contribute to technical documentation
  • Work asynchronously across timezones using GitLab's all-remote culture

Requirements

  • 5+ years of professional frontend development experience
  • Strong proficiency in JavaScript/TypeScript, HTML5, CSS3
  • Experience with Vue.js, React, or similar modern JS frameworks
  • Familiarity with REST APIs, GraphQL, and state management patterns
  • Understanding of CI/CD pipelines and version control with Git
  • Excellent written and verbal communication skills
  • Experience working in a remote or distributed team environment

Nice to Have

  • Experience contributing to open-source projects
  • Knowledge of Ruby on Rails (GitLab's backend framework)
  • Experience with accessibility standards (WCAG 2.1)
  • Familiarity with WebGL, Canvas, or SVG for data visualization

Benefits & Perks

  • Competitive salary: $120,000 – $160,000 per year
  • Equity compensation with stock options
  • Flexible work hours and unlimited PTO
  • $1,500 annual home office stipend
  • Health, dental, and vision insurance
  • Professional development budget ($1,000/year)
  • Annual company retreats and team meetups

Company Profile

GitLab is the most comprehensive DevSecOps platform, delivered as a single application. From planning to production, GitLab helps teams improve their development velocity and reduce costs. With over 30 million registered users and a fully remote workforce spanning 65+ countries, GitLab is at the forefront of the future of work.
